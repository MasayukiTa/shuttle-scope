# Add: サイドバーにロゴ画像表示

Date: 2026-04-23

## 変更

- `shuttlescope/src/App.tsx` の `Sidebar()` 内、左上の "SS" テキストを favicon.png 画像に差し替え。
- ロゴ表示領域のみ、ダークモードでも常に `bg-white` を維持。
  favicon.png が白背景のため、ダークテーマでも浮かないようロゴ帯だけ色を固定。
- ロゴ領域は `border-b border-gray-200` で本体サイドバーと視覚的に分離。

## 動機

Google 検索開始 / SNS 流入増に合わせてブランドアイコンをアプリ内にも露出させる。
サイドバー最上部を「ロゴ帯」として独立させることで、白背景アイコンを
theme に関わらずそのまま流用できる。
